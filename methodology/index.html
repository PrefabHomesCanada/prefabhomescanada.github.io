Our Methodology

How we select listings

We aim to cover a realistic range of options while filtering out listings that lack the minimum information a Canadian buyer needs.

Coverage across common use cases

  • Cabin / cottage: phased builds and seasonal use
  • Backyard suite: where local rules allow
  • Site office / studio: quick deployment scenarios
  • Primary dwelling intent: higher standards, more risk

Minimum bar (or we downgrade heavily)

  • Clear dimensions: footprint + height (as listed)
  • Basic materials: structure/wall system (as listed)
  • Utility clarity: what is included vs “ready for”
  • Shipping reality: delivery/offload clues where possible

We prefer listings with documentation signals

If a seller provides spec sheets, consistent measurements, or detailed inclusions, we treat it as lower risk. If everything is vague, we label it accordingly.

We avoid overpromises

Many online prefab listings use strong language (“luxury”, “eco”, “winter ready”) without measurable specs. We don’t rank based on adjectives.

Practical takeaway: if a listing can’t answer basic questions in writing, plan as if you’re buying a shell and budget for upgrades.

What sources we use

Our guides are built from what the public can realistically access: listing pages, product details, manuals/spec sheets when available, and common Canadian project constraints.

Primary sources

  • Product listing specifications (dimensions, materials, weight)
  • Seller-provided documentation (when supplied)
  • Manuals and installation requirements (when available)

Planning context (Canada)

  • Typical site work categories (foundation, utilities, access)
  • Cold-weather performance considerations
  • Common “missing items” patterns in online prefab homes

How we verify claims

For online prefab homes, the biggest buyer risk is assuming a detail is included when it isn’t. Our verification approach is simple: if it’s not stated clearly, it’s treated as unknown.

What we treat as “verified”

  • Written specs that match across the listing (not contradictory)
  • Clear inclusions list (fixtures vs rough-ins)
  • Electrical approvals stated with identifiable standards (then you still confirm locally)

What stays “unknown” until proven

  • Insulation values, window/door performance, air sealing quality
  • Snow load claims without documentation
  • Permits/engineering “included” unless it’s explicitly a permit package

Our buyer checklist (the questions that save money)

Envelope

  • Insulation type + thickness (walls/roof/floor)
  • Window/door specs (glazing, seals)
  • Air sealing approach (tape, membrane, gasket details)

Structure + roof

  • Frame material + corrosion protection
  • Roof build-up and snow-load documentation
  • Any drawings/spec sheets provided in writing

Utilities

  • Electrical approvals (CSA/UL/ETL), panel details
  • Plumbing: what’s included vs what’s “ready for”
  • Heating plan for your climate zone

Logistics

  • Delivery method and offload requirements
  • Placement needs (crane, forklift, access)
  • Lead time and packaging details

Rule of thumb: If you can’t get written answers, assume higher “integration costs” (trades, upgrades, rework, delays).

Ranking framework

Updates & corrections

Listings can change quickly. When we update a guide, we refresh prices, check whether specs changed, and adjust rankings if the documentation quality shifts.

When we update

  • Major listing changes (specs/photos/inclusions)
  • Large price swings
  • Reader-reported issues that can be verified

How to contact

If you spot an error, contact us with the page URL and the specific section. We prioritize corrections that affect safety, costs, or compliance assumptions.
